What is Machine Learning?
Machine Learning is the study of algorithms which solve complex tasks without giving explicit instructions. Instead of complex instructions, machine learning algorithms simply find patterns in the input data and use those patterns to draw conclusions, all without any direction from a human. Given enough data, machine learning algorithms can theoretically learn anything.
What is Deep Learning?
Deep learning is one particular approach to machine learning. Deep learning algorithms model themselves after the human brain. Like human brains, a neural network is made of many neurons, connected in precise ways. But unlike human brains, each neuron is a simple machine which takes some input and applies a function to it. The power of deep learning comes from layering hundreds of these neurons in precise ways, spotting patterns in the data that simpler machine learning algorithms would fail to detect. Because of their similarity to human brains, deep learning algorithms are also called neural networks.

Deep learning is the best approximation of human intelligence we have so far. A self driving car learning to stay on the road, a smart home assistant learning the sound of your voice, a search engine learning what results are most relevant, and countless other applications -- all of these are complex, interesting applications. And all of these use deep learning.
