Flattening
We've so far had a 2 dimensional matrix of numbers as our image. But for our next step, we're going to need a 1 dimensional list.
The process of such a conversion is called Flattening.



It's a pretty straight forward process. It's just stacking the rows one above another.
Perceptron
Perceptrons are the building blocks of neural networks.
In fact, a neural network is nothing but a multi-layered perceptron.
Still, what is it?
Picture the following -



You have a bunch of inputs x on the left. If you multiply each input by some weights w, and sum them up, you get y.
The inputs here can be the flattened list from earlier.
Now we'll add a couple more bits to this - bias and activation.
Bias

A bias is a constant term added to the weighted sum.

This helps sway the output of the perceptron towards some desired value.
In other words, it makes the perceptron biased.
Activation
An activation function squishes the output of the previous operation.
Neural networks cannot perform well with large values. So the outputs of perceptron are passed through an activation function which brings them down to desirable range.
There are different types of activation functions. Sigmoid and ReLU are popular ones.

To sum up...
Putting all these together gives us a perceptron.



So, a perceptron consists of four parts --
  1. Input values or One input layer
  2. Weights and Bias
  3. Weighted net sum
  4. Activation Function
